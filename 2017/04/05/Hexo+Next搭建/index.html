开始写博客 - Hexo + Next搭建个人博客

发表于 2017-04-05 | 分类于 环境搭建

一直想着写博客，但是在拖延症和懒惰的客观因素下一直给自己找借口。平常学习的一些知识点偶尔也会零零碎碎的用有道、印象笔记、md工具等等记录在本地，但是基本上写了之后过段时间找不到了或者自己也不愿意看。
最近学习的东西也比较零碎，希望可以通过博客他来提升自己的写作能力、总结能力，一个技术点会使用，并不难，但是要做到让别人也可以听懂我们讲的，还是需要一定的技巧和经验。

读书笔记，各路教程，背后原理，新技术等等问题，我希望可以找个东西记下来。为了体现出折腾的本性，CSND、云笔记、博客园太low了（要用自己用Python开发个），样式布局都不太喜欢，看到一些大神们的博客都是高逼格的，使用Hexo写的，我就找了些资料，踩了些坑，最后终于使用Hexo+Next主题搭建个人博客初步完成了。

关于Hexo

  • A fast, simple & powserful blog framework
  • 快速，简单高效的静态博客框架
    • 超快速度: Node.js所带来的超快生成速度，让几百个页面在几秒内瞬间渲染完成
    • 支持Markdown: Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。
    • 一键部署: 只需一条指令即可部署到 GitHub Pages, Heroku 或其他网站。
    • 丰富的插件:Hexo 拥有强大的插件系统，安装插件可以让 Hexo 支持 Jade, CoffeeScript。

关于Nexo

  • NexT is built for easily use with elegant appearance. First things first, always keep things simple
  • NexT 主旨在于简洁优雅且易于使用，所以首先要尽量确保 NexT 的简洁易用性。

  • 安装Git(必须)
      作用：本地的hexo文件提交到github上，安装了Xcode就自带了git，不多说了。
