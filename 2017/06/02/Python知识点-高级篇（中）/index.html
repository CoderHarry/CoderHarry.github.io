1. 装饰器的基本语法

  1. def w1(func): ==> 将w1函数加载到内存

2. 装饰器原理

  1. 执行w1函数，并且将@w1下面的函数作为w1函数的参数，即: @w1等价于w1(f1) 。

  2. 将执行完的w1函数返回值赋值给@w1下面的函数名f1，即将w1返回值重新赋值给f1。

  3. 之后再次调用f1函数的时候其实调用的就是w1返回的那个函数。
